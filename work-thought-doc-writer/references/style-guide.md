# 写作风格

## 基本取向

- 判断明确：文章最终必须形成清楚判断，但不要求第一段直接给结论。
- 阅读路径自然：开头可以从场景、疑问、观察、案例或外部观点进入，让读者跟着作者的思考推进。
- 从素材提炼可迁移内容：不只描述事件，要提炼组织、产品、协作、认知或方法层面的判断、认知、方法或问题意识。
- 语言务实：少官话、少套话、少口号。
- 适合沉淀：文章应适合放入组织知识库、个人工作思考或团队认知材料。

## 阅读推进

- 文章不是汇报材料，不默认使用“结论、原因、方法、总结”的摘要式顺序。
- 开头要让读者愿意往下读，可以先写一个具体场景、一个真实疑问、一个反常观察或一个可理解的案例。
- 判断可以在开头提出，也可以在中段或结尾自然浮现；无论放在哪里，最终都要清楚、可复述。
- 段落之间要有推进关系：从看到什么，到为什么值得想，再到形成什么判断。

## 判断表达

- 每篇文章只围绕一条主线展开，最终形成明确判断。
- 文章需要有可复述的核心判断或金句，但不能为了金句牺牲准确性。
- “不是 A，而是 B”是常用思维结构，不是标题硬模板。可以用于核心判断、段落句、小标题，也可以不用。
- 可以指出问题结构，但不要强迫所有素材都变成“问题诊断”；有些文章只是解释现象、沉淀方法或完成认知转化。
- 不审判具体人，不把个案写成批判文章。

## 类比规则

- 可以使用历史、经济学、管理学、工程、医学、金融、体育等类比。
- 类比必须服务观点迁移，帮助读者把抽象判断理解清楚。
- 不为了显得有文化而堆理论、堆名词、堆外部案例。
- 类比不能引入用户未提供且无法核验的事实；不确定来源时必须说明不知道。

## 标题规则

标题不强制使用反差表达。优先级如下：

1. 言简意赅。
2. 中心思想明确。
3. 方便记忆。
4. 便于知识库检索。
5. 有判断感。

可选标题形式：

- 直接判断型：质量保障的价值在于防止问题复发
- 问题引导型：为什么越努力，复杂工作反而越乱
- 反差判断型：真正的分水岭不是努力，而是顺序
- 概念提炼型：阅读标记是一种认知导航

## 禁用倾向

- 不写官方套话、职场空话、口号式总结。
- 不写成情绪化批评。
- 不写成汇报摘要或结论清单。
- 不把每一段都写成生硬的结论句。
- 不强行把所有素材归结为“结构性问题”。
- 不把文章写成清单大全。
- 不用理论压过素材。
- 不虚构冲突、后果、人物动机或组织背景。
