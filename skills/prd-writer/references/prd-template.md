## PRD 自检清单

输出最终 PRD 前逐项检查：

- [ ] **异常交互**：每个功能的用户流程包含异常路径，不能只写成功路径。
- [ ] **状态机**：所有状态完整列出，每个状态的进入和退出条件明确。
- [ ] **字段规范**：每个输入字段有类型、必填、长度、校验规则、错误提示。
- [ ] **文案规范**：空状态、加载、成功、失败、权限、网络异常六类场景文案齐全。
- [ ] **流程图**：用户流程包含分支和异常分支，不能是单一直线流程。
- [ ] **页面结构**：核心页面、导航层级、页面职责、功能入口和结果页已明确，不能只列功能不列页面。

除非用户要求，否则不要把自检清单作为最终 PRD 的单独章节展示；只用它来补齐 PRD。
